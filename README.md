# Kursovaya
1. Скомпилируйте программу:
    - В терминале, в вашем проекте, выполните команду:
      
      make


2. Запустите сервер:
    - Убедитесь, что сервер, к которому должен подключаться клиент, запущен. Если сервер локальный, он должен слушать на `127.0.0.1` и порту `33333`.

3. Запустите клиентскую программу:
    - Используйте команду для запуска клиента. Например:
      
      ./client_program -a 127.0.0.1 -p 33333 -i input_data.txt -o output_results.bin -c credentials.conf
